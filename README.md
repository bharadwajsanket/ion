# ion

A local-first version control system focused on simplicity and reliability.

## Philosophy

ion is built with a simple goal:

> version control that is predictable, understandable, and does not break.

No unnecessary complexity. No hidden behavior.

## Goals

- Local-first (works fully offline)
- Simple and human-friendly
- Reliable (no corruption, no surprises)
- Fast for small and medium projects

## Commands (Phase 1)
ion init
ion save “message”
ion history
ion restore 
## Status

Early development. Core functionality in progress.
